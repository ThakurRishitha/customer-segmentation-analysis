# Customer Segmentation Analysis

## Project Overview

This project analyzes customer transaction data to identify customer segments, purchasing behavior, and churn risks using Python.

The project includes:

- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- RFM Analysis
- K-Means Clustering
- Customer Segmentation
- Purchase Pattern Analysis
- Customer Retention Analysis
- Churn Risk Identification
- Business Recommendations

---

## Objective

The objective is to help Alfido Tech understand customer behavior by segmenting customers based on purchasing activity and identifying opportunities to improve customer engagement and retention.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## Dataset

The dataset contains customer transaction details including:

- Customer ID
- Purchase Date
- Product Category
- Quantity
- Total Purchase Amount
- Payment Method
- Customer Age
- Gender
- Returns
- Churn

---

## Project Workflow

### 1. Data Cleaning
- Removed missing values
- Removed duplicate records
- Converted Purchase Date to datetime format

### 2. Feature Engineering
Created new features:
- Year
- Month
- Day
- Weekday
- Quarter

### 3. Exploratory Data Analysis
- Sales by Product Category
- Monthly Sales Trends
- Payment Method Distribution
- Gender Distribution
- Customer Age Analysis
- Returns Analysis
- Churn Analysis

### 4. Customer Segmentation

RFM Analysis was performed using:

- Recency
- Frequency
- Monetary

K-Means clustering was applied to segment customers into four groups.

### 5. Business Insights

The analysis identified:
- High-value customers
- Loyal customers
- At-risk customers
- New or inactive customers

---

## Results

The project successfully identified customer segments and purchasing patterns that can support targeted marketing strategies and improve customer retention.

---

## Business Recommendations

1. Introduce a loyalty rewards program.
2. Launch personalized campaigns for at-risk customers.
3. Promote high-performing product categories.
4. Offer seasonal discounts during low-sales periods.
5. Provide exclusive offers to premium customers.
